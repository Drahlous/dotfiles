# Installation

```bash
cd
git clone --recurse-submodules git@github.com:Drahlous/dotfiles.git

cd ~/dotfiles
./bootstrap.sh
```

## Tmux
```
C_a     Prefix

Windows
c       Create new window
d       Delete window
C_h     Move left
C_l     Move right

Pane navigation
-       Split pane horizontal
_       Split pane vertical

h       Move left
j       Move down
k       Move up
l       Move Right

>       Swap next
<       Swap previous

Sessions
C_c     Create new session
$       Rename session

```
